# AI_It_camp

	Программа с одеждой:
Нейронная сеть обучается на 60-ти тысяч картинок распределять их по десяти различным категориям 
(Футболка, брюки, свитер, платье, пальто, сандали, рубашка, кроссовки, сумка,  ботильоны). 
Затем показывает результат на случайных картинках.
Требуемые библиотеки: numpy, matplotlib, keras, tensorflow
	
	Программа с картинками:
Программа, которая может определять различные объекты на фотографии.
Список требуемых библиотек по ссылке: https://imageai.readthedocs.io/en/latest/
Также нужно скачать модель RetinaNet: https://github.com/OlafenwaMoses/ImageAI/tree/master/imageai/Detection
